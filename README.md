
# SmartPet-Memory-Game

It's a memory game.

## Instructions
Click on both cards you'd like to turn, if wrong, cards will turn back. If correct, cards will remain turned. For each try, a specific audio will play.

## Tech used

 1. HTML
 2. CSS
 3. JAVASCRIPT
